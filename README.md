# 🔱 Chiranjeevis Eternal: The Age of Kali

[![Deploy to GitHub Pages](https://github.com/chiranjeevis-eternal/chiranjeevis-eternal.github.io/actions/workflows/deploy.yml/badge.svg)](https://github.com/chiranjeevis-eternal/chiranjeevis-eternal.github.io/actions/workflows/deploy.yml)

> **"When Dharma declines and Adharma rises, the immortals stir from their meditation."**

**Chiranjeevis Eternal** is a high-fidelity Interactive Fiction engine and mythic strategy narrative. Built on the **Ink** narrative language and a custom hardware-accelerated VFX engine, it tells the story of the transition from the decaying Kali Yuga to the dawn of the Satya Dawn.

## 🌌 Key Features

- **Dynamic Narrative Engine:** Powered by `inkjs`, featuring Act-based progression, branching affinity systems, and persistent karma tracking.
- **Cinematic Visuals:** A custom vanilla JavaScript and CSS engine featuring:
  - **Procedural VFX:** Atmospheric ash and ember particles that respond to narrative states.
  - **Depth Layering:** Multi-layered background parallax with Ken Burns animations and radial vignettes.
  - **The Great Chronicle:** An interactive SVG-based map system that tracks your pilgrimage through Shambhala, Mahendra, and beyond.
- **Mythic UI:** A "Golden Ink on Obsidian" aesthetic using modern CSS glassmorphism and typefaces (Inter & Outfit).

## 🛠️ Architecture

- **Narrative:** [Inkle's Ink](https://github.com/inkle/ink) logic.
- **Graphics:** Vanilla HTML5/CSS3 with hardware-accelerated animations.
- **Build System:** [Vite](https://vitejs.dev/) for production-grade builds and asset optimization.
- **Deployment:** GitHub Actions CI/CD to GitHub Pages.

## 🚀 Development

### Prerequisites
- Node.js (v18+)
- [Inklecate](https://github.com/inkle/ink#command-line-ink-compiler) (optional, for manual story compilation)

### Commands
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Compile Story manually
npm run compile-ink

# Build for production
npm run build
```

## 📜 Story Manifest
The Great Chronicle spans six key Acts:
- **Act 0:** The Shambhala Silence (Prologue)
- **Act 1:** Birth of the Avatar
- **Act 2:** The Mahendrapuri Pilgrimage
- **Act 3:** The Trial of Seven Immortals
- **Act 4:** The Obsidian Fortress (Climax)
- **Act 5:** The Mirror of Kali (Resolution)
